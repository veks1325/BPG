플랫포머 게임 보스 패턴 자동 생성 프레임워크 
(Platformer Game Boss Pattern Generator)

> **LLM(Large Language Model)을 활용한 유니티(Unity) 보스 패턴 스크립트 자동 생성 및 난이도 조절 파이프라인**
> *2024 한국디지털콘텐츠학회 추계종합학술대회 발표 논문 기반 프로젝트*

<br/>

## 프로젝트 소개 (Overview)
게임 개발 과정에서 가장 많은 시간과 리소스가 소모되는 **보스 패턴 디자인 및 난이도 조절 과정을 자동화**하기 위해 기획된 프로젝트입니다. 
기존에는 기획자가 패턴을 구상하고 프로그래머가 직접 코드를 작성하며 반복 테스트를 거쳐야 했지만, 본 프로젝트는 **자연어 컨셉 프롬프트를 바탕으로 유니티 엔진에서 즉시 작동 가능한 C# 코루틴(Coroutine) 코드를 자동으로 생성**합니다. 또한, 사용자의 피드백(목표 난이도)에 맞춰 기존 패턴의 속도, 데미지, 정확도 등을 동적으로 재조정합니다.

<br/>

##  주요 기능 (Key Features)

1. **컨셉 기반 프롬프트 생성 (`concept2prompt`)**
   - 짧은 키워드나 아이디어(`concept.txt`)를 입력하면 파인튜닝된 LLM이 구체적인 보스 패턴 기획(프롬프트)으로 구체화합니다.
2. **보스 패턴 코드 자동 생성 (`prompt2code`)**
   - 생성된 기획 프롬프트를 바탕으로 유니티 C# 환경에서 동작하는 코루틴(Coroutine) 기반의 보스 공격 패턴 코드를 즉시 생성하여 파일로 저장합니다.
3. **난이도 재생성 (`regencode`)**
   - CoT(Chain-of-Thought) 기법이 적용된 QA 데이터셋으로 파인튜닝된 모델을 사용하여, 현재 난이도기준으로 패턴을 재 설계한다. 

## 🛠 기술 스택 (Tech Stack)
- **Language:** Python, C#
- **AI / LLM:** OpenAI API (Fine-tuned `gpt-4o-mini` 모델 사용)
- **Game Engine:** Unity
